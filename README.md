# LSEdit

This repository contains a partial code release for **LSEdit**.

The currently available files are intended to provide an early reference for the project structure and selected evaluation scripts. The complete implementation, documentation, and reproducible experimental pipeline will be released after the paper is accepted.

## Current Release

This preliminary release includes:

- Core project files required for the partial implementation.
- Selected PieBench evaluation scripts:
  - `test_flux1LS_piebench.sh`
  - `test_flux2LS_piebench.sh`
- Dependency specification in `requirements.txt`.

Some modules, training/evaluation utilities, experiment configuration files, and full documentation are intentionally withheld at this stage.

## Full Code Release

The full codebase will be uploaded after the paper is accepted. The final release is expected to include:

- Complete source code.
- Full inference and evaluation scripts.
- Reproducible experiment settings.
- Detailed usage instructions.
- Additional documentation and examples.

## Installation

The current partial release provides a preliminary dependency list:

```bash
pip install -r requirements.txt
```

Please note that the current repository may not be fully runnable until the complete code release is available.
